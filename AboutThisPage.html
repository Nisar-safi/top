About This Project

This project provides an overview of active and historical Flipstarters, curated through a manual process due to the decentralized nature of Flipstarter. It is built using SvelteKit by Nisar and was directed and financed by @damascene GitLab.

It uses the Flipstarter Directory Backend for campaign data. The concept is copied from the original deployment and enhanced to add campaign-specific pages and use static pages.
